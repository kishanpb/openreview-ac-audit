# Reports

- plots/ contains SVG plots generated by scripts/enhance_openreview_report_with_plots.py and used by the published post.
- experiment_history.md summarizes the current analysis pass and prior cleanup decisions.
- notion_blog_openreview_ac_overrides.md is an ignored local output generated only when rebuilding or validating the report.
- plots/png/ is an ignored local export folder used only when rebuilding a Notion import package.

Local Notion import copies and zip files are ignored by git because they are generated packaging artifacts.
